How to Use This Guide

Purpose: This framework reorganizes the Top 100 IM conditions by how patients present rather than by disease entity. Use this when you have a presenting symptom, sign, or lab abnormality and need to rapidly generate a differential diagnosis and initial workup plan.

Clinical Approach:

  • Start with the presentation ‚Üí Select the presenting feature (e.g., chest pain, dyspnea, altered mental status)
  • Apply branch points ‚Üí Use clinical discriminators to narrow your differential (e.g., "ST elevation present?" "Fever?" "Focal deficits?")
  • Identify must-not-miss diagnoses ‚Üí High-acuity conditions that require urgent recognition and treatment
  • Order initial workup ‚Üí Evidence-based first tests to differentiate causes
  • Cross-reference ‚Üí Click through to detailed condition entries in the main guide for full management

Key Features:

Differential Diagnosis Framework

üîÄ Key Branch Points / Clinical Discriminators

Use these clinical features to narrow your differential:

  • ST elevation present vs absent
  • Troponin elevation
  • Associated dyspnea/hypoxia
  • Hemodynamic stability
  • Pleuritic vs non-pleuritic

  • #7: Cardiac Tamponade
  • #19: Tension Pneumothorax

Thrombocytopenia (Platelets <150k)

Differential Diagnosis Framework

üîÄ Key Branch Points / Clinical Discriminators

Use these clinical features to narrow your differential:

  • Severity (<50k, <20k, <10k)
  • Bleeding present
  • Isolated vs with anemia/leukopenia
  • Recent medications/heparin
  • Evidence of consumption (DIC, TTP)

‚ö†Ô∏è MUST NOT MISS Diagnoses

High-acuity or rapidly progressive conditions requiring urgent recognition:

  • #79: ITP/HIT/TTP
  • #78: DIC
  • #77: Hemolytic Anemia with TTP/HUS
  • #30: Neutropenic Fever if pancytopenia

üî¨ Initial Diagnostic Workup

Essential tests to differentiate causes:

  • CBC with peripheral smear
  • PT/INR, PTT, fibrinogen (DIC screen)
  • LDH, indirect bilirubin, haptoglobin
  • BMP (assess renal function for TTP/HUS)
  • Medication review (especially heparin)
  • HIT antibody if heparin exposure
  • Consider HIV, HCV testing
  • Bone marrow biopsy if persistent/unexplained
